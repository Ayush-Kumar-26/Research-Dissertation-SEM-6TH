# Dissertation Project: Data Analysis using Excel

## Overview
This repository contains my dissertation project focused on analyzing data using Microsoft Excel. The objective of this study is to extract meaningful insights and support data-driven decision-making.

## Dataset
The dataset used in this project is included in this repository in Excel format.

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP
- Data Cleaning
- Data Visualization (Charts & Graphs)

## Key Insights
- Identified trends and patterns in the dataset
- Calculated mean values (e.g., average rating ~3.8)
- Used Pivot Tables for summarizing large data
- Applied VLOOKUP for efficient data retrieval

## Files Included
- `Excel_Data_Analysis_Pivot_VLOOKUP.xlsx` → Contains raw data, analysis, and visualizations

## Conclusion
This project demonstrates practical skills in data analysis and visualization using Excel, helping in better interpretation of structured data.
